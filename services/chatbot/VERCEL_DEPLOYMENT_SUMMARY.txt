DEPLOYMENT INSTRUCTIONS
================================================================================

OPTION 1: Via Vercel Dashboard (Recommended for first-time)
---------------------------------------------------------------------------
1. Go to https://vercel.com/new
2. Import your Git repository
3. Set root directory to: services/chatbot
4. Vercel will auto-detect Vite
5. Click "Deploy"
6. Add environment variables in Settings
7. Add custom domain (optional)

OPTION 2: Via Vercel CLI
---------------------------------------------------------------------------
1. Install CLI: npm i -g vercel
2. Navigate: cd services/chatbot
3. Deploy: vercel
4. Follow prompts
5. Set environment variables: vercel env add VITE_API_URL
6. Deploy to production: vercel --prod

OPTION 3: Via Git Auto-Deploy
---------------------------------------------------------------------------
1. Connect repository to Vercel
2. Push to main branch: git push origin main
3. Vercel automatically deploys
4. Preview deployments for other branches

POTENTIAL ISSUES AND SOLUTIONS
================================================================================

Issue 1: Blank Page After Deploy
Cause: Missing SPA rewrite rules
Solution: vercel.json already includes rewrites (✅ Fixed)

Issue 2: API Calls Fail
Cause: Incorrect VITE_API_URL or CORS issues
Solution: Verify environment variables in Vercel Dashboard

Issue 3: Environment Variables Not Working
Cause: Missing VITE_ prefix
Solution: Always use VITE_ prefix (✅ Documented)

Issue 4: Build Fails on Vercel
Cause: TypeScript errors or missing dependencies
Solution: Check build logs, ensure package.json is complete

Issue 5: Assets Return 404
Cause: Incorrect asset paths
Solution: Vite handles this automatically, verify build output
